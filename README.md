### Инструкция по поднятию бота:

Заполнить файл с перемнными огружения *".env"*.
Находясь в корневой директории проекта выполнить команду:

```sh
sudo docker-compose up
```

### Стек технологий:

* PostgreSQL
* asyncpg
* google-api-python-client
* google-auth
* Docker
